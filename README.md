有著勇氣、勇敢的意思。  

> 「立慬者非學鬥爭也，慬立而生不讓。」-- 《淮南子．說山》
   
意味著學會勇敢不是為了爭搶，但是勇敢卻是伴隨著不謙讓而生。

那麼如何成為真正勇敢的人，是很重要的課題。

就已經是難能可貴的了。
